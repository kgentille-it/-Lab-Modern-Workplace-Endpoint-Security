# -Lab-Modern-Workplace-Endpoint-Security
Ce projet démontre la mise en place d’un environnement Modern Workplace sécurisé basé sur Microsoft Entra ID et Intune, suivant une approche Zero Trust.

Compétences mises en œuvre

Identité : Microsoft Entra ID, gouvernance par groupes

Sécurité : Accès conditionnel (MFA, géolocalisation, device compliant)

Endpoint Management : Enrôlement Intune, conformité Windows 10/11

Chiffrement : BitLocker, Secure Boot, TPM

Troubleshooting : analyse et résolution de non-conformité MDM
